Υλοποίησα αυτήν την εργασία με χωρητικότητες (παρόλο που δεν είχα χωριτηκότητες στην εκφώνηση) μιας και έτσι ήταν και η προηγούμενη εργασία και μου ήταν εύκολο(μπαινουν κατα κύματα και οχι ενας ενας).Ώστοσο ορίζονται by deafult με μεγάλες τιμές για να μην επηρεάζουν 
το αποτέλεσμα.Στην περίπτωση που κάποιος μείνει εκτός τάξης,έχω ένα for loop στην main που να λειτουργεί το κτήριο και άλλες φορές ώστε να βγουν όλοι από αυτό.Συγκεκριμένα όταν βγάζω έναν μαθητή και έχει μείνει κάποιος έξω από την τάξη,τότε τον βάζω μέσα.
Δηλαδή όταν έχω overflow για μια τάξη ,για κάθε έναν που βγάζω ,βάζω έναν.Χρησιμοποιώ κληρονομικότητα για τις κλάσεις Student/Teacher με parent class την Person.Επίσης χρησιμοποιώ virtual ορισμό συναρτήσεων στην mother class
για να ορίσω τις μεθόδους των κλάσεων.Με παρόμοιο τρόπο ορίζονται οι κλάσεις του κτηρίου .Αν και δεν έχω φτάξει συνάρτηση που να εκτυπώνει το building consists of ,εκτυπώνω σε κάθε μετάβαση τα κατάλληλα μηνύματα για να βλέπω που είναι
ο καθένας σε κάθε στάδιο.Και οι constructors / destructors εκτυπώνουν μηνύματα.Το concept μου είναι να μπαίνουν πρώτα οι καθηγητές και μετά οι μαθητές στις κλάσεις(και οι teachers στην υλοποίηση μου κάνουν την πορεία των .Οι destructors μου δεν λειτουργούν πολύ καλά(βγάζουν segmetation fault άμα αφήσω όλα τα delete).
Το πρόγραμμα δοκιμάστηκε για 11 και 31 μαθητές.Λογικά λειτουργεί για οποιοδήποτε αριθμό.Οι δάσκαλοι by default είναι 18.
g++ -g -o  myprogram class4.cpp
./myprogram 3 5 2 5 11
